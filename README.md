\# AméricaFitness



Plataforma profesional para cadenas de gimnasios. App móvil con check-in QR, reservas de clases, rutinas personalizadas, nutrición integrada y sistema de pagos.



\## Arquitectura



\- \*\*Backend\*\*: FastAPI + MySQL + JWT

\- \*\*App Móvil\*\*: Flutter

\- \*\*Infraestructura\*\*: Docker + GitHub Actions (próximamente)
